# Pleasure Live

Welcome to Pleasure Live, a live streaming application built with the MERN stack (MongoDB, Express, React, Node.js) and WebSockets. This project aims to provide a platform for users to broadcast live video, interact with viewers in real-time, and discover new content.

## Features (MVP)

*   **User Authentication**: Secure user registration and login.
*   **Live Streaming**: Users can start and stop live streams.
*   **Real-time Chat**: Viewers can chat with the streamer and other viewers during a live broadcast.
*   **Stream Discovery**: Users can browse and discover active live streams.

## Project Structure

*   `/frontend`: Contains the React client application.
*   `/backend`: Contains the Node.js/Express server and WebSocket implementation.

This project is currently under development.
